Especificidade

Hierarquicamente o estilo a seguir sera sempre a seguinte sequencia:

  1. Inline
  2. Folha de style
  3. ID
  4. Classe
  5. Somatoria de Especificidade
  6. Tag

Importante:

Caso precise explicitar que algum estilo deve ser considerado independente de hierarquia devi usar a tag !important asim sobrepondo qualquer especificidade.

Não é um boa pratica o uso do memso demasiadamente pois tira o real ojetivo do mesmo!
